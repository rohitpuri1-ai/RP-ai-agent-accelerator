# AI Agent Kick Starter

## Course Description

The **AI Agent Kick Starter** is a foundational learning program designed for DSM students to develop a clear, practical understanding of how AI agents work inside real automation systems.

This program intentionally avoids complexity.

Instead of teaching advanced multi-agent architectures upfront, it focuses on **mental models, data flow, and execution logic** that power every AI agent, regardless of scale.

Each workflow in this kick starter is:
- Small and isolated
- Built around a single core concept
- Designed to be read, modified, and reused
- Meant to build confidence, not dependency

The objective is not to memorize nodes or tools.

The objective is to help students **think in workflows**:
- How data enters a system
- How decisions are made
- Where AI adds value
- How outputs reach real users

By completing this kick starter, students should be able to:
- Understand the anatomy of an AI agent
- Modify existing workflows without breaking them
- Decide when to use AI vs deterministic logic
- Build and deploy simple agents independently
- Transition smoothly into advanced agentic systems

This course acts as the **structural foundation** for all advanced AI agent programs at DSM.
