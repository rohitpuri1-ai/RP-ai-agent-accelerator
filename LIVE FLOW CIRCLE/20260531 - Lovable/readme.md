## The Prompt

Copy and paste this into Lovable to build the application:

```
Build a modern Kanban Task Management web application.

Requirements:

1. Authentication
   - Users can sign up using email and password.
   - Users can log in and log out.
   - Each user can only view and manage their own tasks.

2. Task Management
   - Users can create tasks.
   - Each task should have:
     - Title
     - Description
     - Priority (Low, Medium, High)
     - Due Date
     - Status

3. Kanban Board
   Create four Kanban columns:
   - Todo
   - In Progress
   - On Hold
   - Completed
   Users should be able to drag and drop tasks between columns.

4. Database
   - Store all tasks in the database.
   - Associate every task with the logged-in user.
   - Persist all data between sessions.

5. Dashboard
   Display:
   - Total Tasks
   - Completed Tasks
   - Pending Tasks
   - Overdue Tasks

6. Design
   - Modern SaaS design
   - Clean layout
   - Responsive mobile-friendly interface
   - Professional dashboard appearance

7. Security
   - Ensure users can only access their own data.
   - Generate all database tables and relationships automatically.
   - Deploy-ready application.
```
